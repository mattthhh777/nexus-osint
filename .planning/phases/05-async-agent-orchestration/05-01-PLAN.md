```python
# api/orchestrator.py
"""
NexusOSINT — Task Orchestrator (api/orchestrator.py)
Concurrent OSINT module execution with dual semaphore control and queue bridge.

Design (per Phase 05 decisions D-01 through D-06):
  - Global asyncio.Semaphore(5): hard ceiling for ALL concurrent module tasks
  - OathNet asyncio.Semaphore(3): limits OathNet modules to max 3 of the 5 global slots
  - asyncio.Queue bridge: each module result pushed as (name, result_or_exception) tuple
  - Tracked create_task + registry (NOT TaskGroup — D-02/D-03: yield inside TaskGroup is impossible)
  - _guarded() wrapper: catches per-module exceptions, pushes error to queue, never crashes orchestrator

Usage:
    orchestrator = TaskOrchestrator()
    orchestrator.submit("breach", coro, is_oathnet=True)
    orchestrator.submit("sherlock", coro, is_oathnet=False)
    async for name, result in orchestrator.results():
        if isinstance(result, Exception):
            handle_error(name, result)
        else:
            handle_result(name, result)
"""
```

Class structure:

1. `__init__(self, max_concurrent: int = 5, max_oathnet: int = 3)`:
   - `self._global_sem = asyncio.Semaphore(max_concurrent)`
   - `self._oathnet_sem = asyncio.Semaphore(max_oathnet)`
   - `self._registry: dict[str, asyncio.Task] = {}`
   - `self._result_queue: asyncio.Queue[tuple[str, Any]] = asyncio.Queue()`
   - `self._expected: int = 0` (count of submitted modules, used by consumer to know when done)
   - Logger: `logging.getLogger("nexusosint.orchestrator")`

2. `submit(self, name: str, coro: Coroutine, *, is_oathnet: bool = False) -> None`:
   - Creates a task via `asyncio.create_task(_guarded(name, coro, is_oathnet), name=f"module-{name}")`
   - Registers task in `self._registry[name] = task`
   - Increments `self._expected += 1`

3. `async _guarded(self, name: str, coro: Coroutine, is_oathnet: bool) -> None`:
   - If `is_oathnet`: acquire BOTH `self._oathnet_sem` AND `self._global_sem` (OathNet first, then global — consistent ordering prevents deadlock)
   - If not `is_oathnet`: acquire only `self._global_sem`
   - Inside semaphore(s): `try: result = await coro; await self._result_queue.put((name, result))` / `except Exception as exc: await self._result_queue.put((name, exc)); logger.warning("Module '%s' failed: %s", name, type(exc).__name__)`
   - `finally:` deregister from `self._registry.pop(name, None)`

4. `async def results(self) -> AsyncGenerator[tuple[str, Any], None]`:
   - Yields `(name, result_or_exception)` tuples from `self._result_queue`
   - Loops `self._expected` times calling `await self._result_queue.get()`
   - Yields each item
   - After exhausting all expected results, returns (generator exits)

5. `@property active_count(self) -> int`: returns `len(self._registry)`

6. `async def cancel_all(self) -> None`:
   - Iterates `list(self._registry.items())`
   - Cancels each non-done task
   - Awaits `asyncio.gather(*tasks, return_exceptions=True)` to ensure cleanup
   - Clears registry
   - Drains any remaining items from result_queue

Important implementation details:
- Do NOT use `asyncio.TaskGroup` anywhere (per D-02, D-03)
- The `results()` async generator is what `_stream_search` will iterate over to yield SSE events
- Semaphore acquisition order for OathNet: `_oathnet_sem` first, then `_global_sem` — ALWAYS this order to prevent deadlock
- No generic `except Exception:` in the class body outside `_guarded` — use specific types where possible
- Use `from __future__ import annotations` at file top
- Follow CONVENTIONS.md: section banners `# -- Section Name ---`, logger as `nexusosint.orchestrator`
