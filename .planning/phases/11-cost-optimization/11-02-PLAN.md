    2. Find the async function that checks platforms (likely `search_username` or `_check_platform`).
       Replace aiohttp.ClientSession usage with httpx.AsyncClient:

       aiohttp pattern:
       ```python
       async with aiohttp.ClientSession() as session:
           async with session.get(url, timeout=...) as resp:
               status = resp.status
               text = await resp.text()
       ```

       httpx replacement:
       ```python
       async with httpx.AsyncClient(timeout=httpx.Timeout(15.0), follow_redirects=True) as client:
           resp = await client.get(url)
           status = resp.status_code  # NOTE: httpx uses status_code, not status
           text = resp.text  # NOTE: httpx .text is a property, not coroutine
       ```

    3. Key API differences to handle:
       - `resp.status` (aiohttp) -> `resp.status_code` (httpx)
       - `await resp.text()` (aiohttp) -> `resp.text` (httpx, sync property)
       - `await resp.json()` (aiohttp) -> `resp.json()` (httpx, sync method)
       - `aiohttp.ClientTimeout(total=N)` -> `httpx.Timeout(N)`
       - `aiohttp.ClientError` -> `httpx.HTTPError`

    4. Error handling: replace `except aiohttp.ClientError` with `except httpx.HTTPError`
    5. If there is a `except asyncio.TimeoutError` keep it (httpx can also raise this)
  </action>
