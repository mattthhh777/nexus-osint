**Part A — Add sanitizeImageUrl() to static/js/utils.js:**

Insert the following function AFTER the `escAttr()` function (after line 52, before the `// ── Toast notification ──` banner):

```javascript
// ── URL sanitization ──
function sanitizeImageUrl(url) {
  if (!url) return '';
  try {
    const parsed = new URL(url);
    return parsed.protocol === 'https:' ? url : '';
  } catch (e) {
    return '';
  }
}
```

Use 2-space indentation, single quotes for strings — per project conventions.

**Part D — Add post-innerHTML event listener attachment in renderExtras():**

AFTER the `el.innerHTML = parts.length ? ...` assignment at line 702, add:

```javascript
// Attach error handlers to images with data-fallback (replaces inline onerror)
el.querySelectorAll('img[data-fallback]').forEach(img => {
  img.addEventListener('error', function() {
    this.style.display = 'none';
    const sibling = this.nextElementSibling;
    if (sibling) sibling.style.display = 'flex';
  });
});
```

This MUST be placed AFTER the innerHTML assignment — event listeners on elements not yet in the DOM are lost when innerHTML is replaced.
