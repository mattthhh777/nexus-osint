<objective>
Wire the Phase 16 backend primitives (Plan 01 config + budget; Plan 02 sherlock engine) into the live request flow:

1. **Username hardening at the boundary** — Pydantic `SherlockUsernameRequest` rejects garbage, control chars, URL injection (D-H8/D-H9). Lives in `api/schemas.py` (leaf — `re + pydantic` only).
2. **Budget circuit breaker** in the SSE service layer — checks `api.budget.is_hard_limit_exceeded()` BEFORE invoking Sherlock; emits a module_error SSE event with `retry_after=86400` instead of silently dropping (D-H12).
3. **SSE serializer extension** — adds `state`, `confidence`, `likely`, `proxy_used` to the Sherlock event (D-13/D-14). `negative_markers` and per-signal raw scores NEVER serialized (D-H2/D-H3).
4. **Admin /health Thordata metrics** — surfaces `bytes_today_mb`, `requests_today`, `budget_remaining_pct`, `proxy_active` as a nested object on the existing `/health` response, returned only when caller is admin (D-19/D-H14).
5. **Lifespan proxy health check** — `GET https://api.ipify.org` through Thordata on startup, non-blocking, sets `api.budget._proxy_active`. Failure → WARNING log + degrade to no-proxy mode (D-07).

Purpose: Plan 04 (frontend + e2e) consumes the SSE contract this plan publishes. Without this plan, the backend has the engine but no caller — and Sherlock would still receive raw user input bypassing the validator.

Output: 5 backend files modified, 1 unit test file new (~80 LOC), 1 integration test file new (~250 LOC).

    async def _thordata_startup_check() -> None:
        """Phase 16 D-07: non-blocking proxy reachability check.

        Sets api.budget._proxy_active. Failure does NOT crash the app —
        Sherlock falls back to direct DO IP path (D-H5: log masked URL only).
        """
        if not THORDATA_PROXY_URL:
            logger.info("Thordata proxy unset — Sherlock will use direct DO IP")
            _budget._proxy_active = False
            return

        masked = _masked_proxy_log(THORDATA_PROXY_URL)
        try:
            async with httpx.AsyncClient(

                timeout=10.0,
                verify=False,
            ) as client:

            logger.info("Thordata proxy OK | proxy=%s exit_ip=%s", masked, exit_ip)
