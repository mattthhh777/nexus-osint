UptimeRobot — External Monitor (Recommended)
Free · 5min checks

UptimeRobot monitors your site from multiple global locations — independent of your VPS. If the VPS goes down, you still get alerted.

Setup Steps
1.
Go to uptimerobot.com → Create free account
2.
Add New Monitor → Type: HTTPS
3.
URL: https://nexusosint.uk/health
4.
Keyword: "ok" (validates the response, not just HTTP 200)
5.
Alert contacts: Email + Telegram (recommended)
6.
Add second monitor for https://nexusosint.uk/api/login (POST endpoint alive check)
Create UptimeRobot Account →
Uptime Kuma — Self-Hosted (Advanced)
Requires VPS config

Uptime Kuma runs alongside your app, monitors all internal services (including the database and Redis if added later), and provides a beautiful status dashboard.

Deploy Command (VPS)
docker run -d --restart=always -p 3001:3001 -v uptime-kuma:/app/data --name uptime-kuma louislam/uptime-kuma:1

After running, access at http://146.190.142.50:3001 — create account, add monitors for each service endpoint.
